# 🍽️ Recipe App (API Evolution Project)

This is a **small but practical Recipe App project** that demonstrates the evolution of building and improving
a REST API using JavaScript (Node.js).

The project focuses on how an API is structured and improved step by step, 
from basic data handling to a more organized and scalable architecture.

## 🚀 Project Purpose

The main goal of this project is to learn and demonstrate:

- How to build a simple REST API using JavaScript
- How to organize routes, controllers, and services
- How APIs evolve from basic to more structured versions
- How to handle CRUD operations for recipes
- How to work with data using JSON

---

## 📈 API Evolution Stages

This project shows progressive improvements in API design:

### 1️⃣ Basic Version
- Simple in-memory data (array)
- Direct route handling
- No separation of concerns

### 2️⃣ Improved Structure
- Separation of logic from routes
- Better code readability

### 3️⃣ Scalable Version
- Reusable functions/services
- Cleaner API design
